# TypeScript Starter Template

A clean, modern, and highly performant Node.js starter template. Built with pure TypeScript, `tsup` for blazing-fast builds, and `vitest` for testing. This template provides a solid foundation for building scalable backend applications, CLI tools, or publishable npm packages.

## ✨ Features

- **Pure Node.js & TypeScript**: No frontend frameworks, just pure backend/Node.js logic.
- **Blazing Fast Builds**: Uses `tsup` (powered by esbuild) for rapid compilation.
- **Path Aliases**: Pre-configured `@/*` aliases for clean, maintainable imports.
- **Environment Validation**: Secure and strictly typed environment variables using `zod` and `dotenv`.
- **Lightning Fast Testing**: Unit testing configured out of the box with `vitest`.
- **Code Quality**: Pre-configured with ESLint (Flat Config), Prettier, Husky, and lint-staged.
- **Automated Releases**: Versioning, tagging, and changelog generation using `release-it`.
- **ESM First**: Fully configured for modern ECMAScript Modules (`"type": "module"`).

## 🚀 Quick Start

### 1. Prerequisites

- Node.js 18+
- pnpm (recommended)

### 2. Installation

```bash
# Clone the repository
git clone https://github.com/sikandarmoyaldev/typescript-starter-template.git
cd typescript-starter-template

# Install dependencies
pnpm install
```

### 3. Environment Variables

Copy the example environment file and update the variables:

```bash
cp .env.example .env.local
```

### 4. Usage

```bash
# Run the development server with hot-reloading
pnpm dev

# Build for production
pnpm build

# Start the compiled production build
pnpm start

# Run tests
pnpm test
```

## 🛠️ Development Scripts

- `pnpm dev`: Starts the development server with `tsx` (watches for file changes).
- `pnpm build`: Compiles TypeScript to JavaScript using `tsup` and generates `.d.ts` declaration files.
- `pnpm start`: Runs the compiled production code from the `dist` folder.
- `pnpm test`: Runs unit tests using Vitest.
- `pnpm lint`: Checks code for errors using ESLint.
- `pnpm format`: Formats all code using Prettier.
- `pnpm release`: Bumps version, updates CHANGELOG, and creates a GitHub release.

## 📦 Publishing as an npm Package

This template is pre-configured to be published as an npm package. The `package.json` includes the correct `main`, `types`, and `exports` fields.

Simply run `pnpm release` to automatically bump the version, generate a changelog, and publish to npm (ensure you are logged in via `npm login`).
